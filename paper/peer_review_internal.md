# 内部同行评审报告

**论文标题：** Lattice Green Functions in $d \geq 4$: A Domb Number Decomposition and the Absence of Classical Closed Forms

**作者：** Jian Zhou

**目标期刊：** Journal of Physics A: Mathematical and Theoretical

**评审日期：** 2026-03-18

**审稿人专业领域：** 格点统计力学、特殊函数、Calabi–Yau 几何、数论

---

## 一、论文概述

本文研究 $d \geq 4$ 维简单立方格点的格点 Green 函数 $G_d(0)$。主要贡献有三：(1) 证明四维随机游走系数的 Domb 数分解 $c_n = \binom{2n}{n} D_n$；(2) 用 Bessel 积分法将 $G_4(0)$ 等计算到 1000/500 位小数；(3) 系统性地用 PSLQ 算法排除 $G_4(0)$ 的各种经典闭合表达式。

论文结构清晰，数学表述整体专业。但在原创性声明、证明严谨性和若干技术细节上存在需要认真处理的问题。

---

## 二、逐项评审

### 2.1 数学正确性

#### (A) 定理 3.1（Domb 分解）— 证明正确 ✓

核心分解 $c_n = \binom{2n}{n} D_n$ 的证明逻辑正确。我逐步验证了关键代数步骤：

- 比值 $\frac{\binom{2n-2}{n-1}}{\binom{2n}{n}} = \frac{n}{2(2n-1)}$ ✓
- 比值 $\frac{\binom{2n-4}{n-2}}{\binom{2n}{n}} = \frac{n(n-1)}{4(2n-1)(2n-3)}$ ✓
- 代入化简后得到 $n^3 D_n = (2n-1)(10n^2 - 10n + 4) D_{n-1} - 64(n-1)^3 D_{n-2}$，经 index shift $n \to n+1$ 确实还原 Domb 递推 (2.5) ✓
- 初始条件 $c_0 = 1, c_1 = 8$ 匹配 ✓

数值验证对 $n = 0, \ldots, 7$ 全部通过。三项递推 (1.2) 对 $n = 2, \ldots, 7$ 验证正确。

#### (B) 命题 3.2（积分表示）— 有严重问题 ✗

**问题一：证明中缺少 $z^n$ 因子。** 证明写道：
$$G_4(0) = \lim_{N \to \infty} \Bigl( \sum_{n=0}^N c_n - \frac{A_4}{\sqrt{\pi n}} \Bigr)$$
但根据引言定义，$G_4(0) = P_4(1/64) = \sum_{n=0}^\infty c_n / 64^n$，此处遗漏了 $(1/64)^n$ 因子。这是证明中的明确错误。

**问题二：不必要的重正化。** 论文定义
$$G_D(z) = \lim_{N \to \infty} \Bigl( \sum_{n=0}^N D_n z^n - B_D \sqrt{N} \Bigr),$$
暗示级数发散需要重正化。但我的计算表明 Domb 数的渐近增长率为 $D_n \sim C \cdot 16^n / n^{3/2}$（从递推特征方程 $r^2 - 20r + 64 = 0$ 得到主根 $r = 16$），因此 Domb 生成函数 $g(z) = \sum D_n z^n$ 的收敛半径为 $R = 1/16$。在积分 (3.6) 中，$z = x/16 \in [0, 1/16]$，在整个积分域上级数收敛（边界 $z = 1/16$ 处因 $D_n / 16^n \sim C/n^{3/2}$ 也收敛）。**重正化完全不必要**，反而引起混乱。

**正确的命题陈述应为：**
$$G_4(0) = \frac{1}{\pi} \int_0^1 \frac{g_D(x/16)}{\sqrt{x(1-x)}} \, \mathrm{d}x,$$
其中 $g_D(z) = \sum_{n=0}^\infty D_n z^n$ 是 Domb 数的普通生成函数（在 $|z| \leq 1/16$ 上收敛），证明只需：
1. 使用 beta 积分 $\binom{2n}{n} = \frac{1}{\pi} \int_0^1 \frac{(4x)^n}{\sqrt{x(1-x)}} \mathrm{d}x$ ✓
2. 代入 $G_4(0) = \sum c_n / 64^n = \sum \binom{2n}{n} D_n / 64^n$
3. 交换求和与积分（由控制收敛定理保证，因为 $\sum D_n / 16^n < \infty$ 且 $\int_0^1 1/\sqrt{x(1-x)} \, \mathrm{d}x = \pi < \infty$）

#### (C) Domb 递推系数 — 注记 2.4 的声明需要更精确的引用

注记声称 Domb 递推 (2.5) 的系数为 $(10n^2 + 10n + 4)$，而非某些文献（如 OEIS A002895）所述的 $(10n^2 + 10n + 3)$。我验证了 $(10n^2 + 10n + 4)$ 对 $n = 1, \ldots, 5$ 均正确，$(10n^2 + 10n + 3)$ 代入会产生非整数结果。但论文应给出具体引用来源（OEIS 哪个公式编号、哪篇论文），而非笼统地说 "as sometimes stated in the literature"。

#### (D) 方程 (1.3) 中的变量错误 ✗

论文称 Fuchsian ODE 使用变量 "$u = z^2$"，但奇异点 $u = 0, 1, 4, \infty$ 与此不一致。验证表明：

| 变换 | $z = 1/64$ 映射到 | $z = 1/16$ 映射到 | 是否匹配 |
|------|-------------------|-------------------|----------|
| $u = z^2$ | $u = 1/4096$ | $u = 1/256$ | ✗ |
| $u = 64z$ | $u = 1$ | $u = 4$ | ✓ |

正确的变量变换应为 $u = 64z$（或等价表述）。这是明确的笔误。

### 2.2 新颖性

#### (A) 核心恒等式 $c_n = \binom{2n}{n} D_n$ — 非新结果 ⚠️

**这是本文最严重的问题。** 恒等式 $c_n = \binom{2n}{n} D_n$（即 OEIS 中 A002896(n) = A000984(n) × A002895(n)）已记录在 OEIS A002896 的公式列表中。论文将此作为 "Theorem 3.1 (Domb Decomposition)" 提出，标题中也以此为核心贡献之一，但未引用任何先前来源。

**建议处理方式：**
- 明确引用 OEIS 及最早发现此恒等式的文献
- 将贡献重新定位为：提供了基于递推验证的严格证明、Hadamard 乘积的 AESZ 算子解释、以及积分表示
- 修改摘要和引言中的表述，避免宣称发现恒等式

#### (B) Hadamard 乘积结构 $\mathrm{AESZ}\,\#16 = \mathrm{Had}(1/\sqrt{1-4z},\, \mathrm{AESZ}\,\#34)$

Calabi–Yau 算子的 Hadamard 乘积分解是该领域已知的概念（参见 Bostan et al. 2013，论文中已引用）。但这个具体的分解关系在文献中是否被明确陈述过，需要作者进一步调查和说明。如果确实未被明确提出，这可以作为一个有价值的观察。

#### (C) PSLQ 排除程序 — 有一定新意

系统性地排除 $G_4(0)$ 的多种闭合形式候选具有价值。Guttmann (2009) 和 Zucker (2011) 已进行过部分排除，但本文的排除范围更广（包括 $\Gamma(p/q)$ 到 $q \leq 24$、Watson 积分、维度间比率等）。这是论文最有实质贡献的部分。

#### (D) 高精度计算

如果 $G_4(0)$ 的 1000 位精度是首次计算（此前最高精度约 12 位，来自 Guttmann 2009），这是对社区有用的贡献。但论文应明确声明此前的精度记录是多少。

### 2.3 写作质量

#### 结构
论文结构合理：引言 → 定义 → Domb 分解 → 计算 → PSLQ → CY 几何 → 结论。各节逻辑衔接自然。

#### 需改进之处

1. **缺少图表。** 作为计算数学论文，完全没有图形令人遗憾。建议添加：
   - Bessel 积分被积函数 $e^{-4t} [I_0(t)]^4$ 的衰减图
   - Richardson 外推的收敛曲线
   - $G_d(0)$ 随维度 $d$ 的变化趋势图
   - PSLQ 搜索中最小关系范数随精度的变化（展示排除的统计置信度）

2. **命题 3.2 的证明。** 如上所述，证明有错误且不完整，需要重写。

3. **第 6 节（CY 几何）过于推测性。** 关于 paramodular forms 的讨论主要是引用性质的，缺乏新计算或新结果。猜想 6.3 尤其模糊（"simple algebraic combination"）。建议要么加强这一节（提供具体的数值证据），要么将其缩短为一个 Remark。

4. **引言重复。** Joyce (1994, 2002) 在引言第 2 段和第 3 段被重复提及，措辞相似。

5. **符号一致性：AESZ 编号。** 论文中 $\mathrm{AESZ}\,\#16$ 既指微分算子，又指生成函数（方程 3.5），这是不准确的混用。AESZ 编号指的是 Fuchsian 算子，不是其解。应明确区分。

### 2.4 技术细节

#### 高精度计算
- 内部精度 1200 位用于输出 1000 位结果，符合标准做法 ✓
- 从 10,000 项递推进行 60 阶 Richardson 外推，结果与 Bessel 积分一致到 46 位 — 这里 46 位是 Richardson 方法的精度上限，说明递推 + 外推只能达到中等精度（相比 1000 位），方法间的交叉验证受限。论文应讨论为何 Richardson 只达到 46 位（例如，外推阶数与截断阶数的匹配问题）。
- 未提供代码或补充材料，可重复性仅依赖方法描述。建议提供 Python 脚本。

#### PSLQ 参数
- 精度范围 "400–800 位"过于笼统。对于不同维度的 PSLQ 向量，所需精度不同。
- Gamma 值测试的 PSLQ 向量维度约 183（$q \leq 24$ 的所有 $\Gamma(p/q)$），在系数界 $M = 10000$ 下需要精度 $p > 183 \times \log_{10}(10000) + \text{margin} = 732+$ 位。800 位精度刚好够用，400 位则**不足以完成此测试**。
- 论文未讨论 Gamma 函数间的代数关系（反射公式、乘法公式）如何降低有效维度。利用这些关系可以减少独立参数数量，提高排除的可靠性。
- 论文未给出 PSLQ 返回的最小关系范数下界，这是量化排除置信度的标准信息。

#### 附录数据精度
论文声称 $G_4(0)$ 计算到 1000 位小数，但附录数据实测为 **996 位**（小数点后），差 4 位。虽是小事，但应精确。

## 三、总体评估

### 优点

1. **主题重要且有吸引力。** 高维格点 Green 函数是统计力学和数论的经典问题，$d \geq 4$ 的情况确实缺乏闭合表达式，研究价值明确。

2. **PSLQ 排除程序系统全面。** 排除候选列表涵盖了合理的闭合形式假设，对社区有直接参考价值。这是论文最有实质性的贡献。

3. **高精度数值数据。** 1000 位精度的 $G_4(0)$ 值（如为首次计算）对后续研究者有用。

4. **CY 几何联系。** 虽然讨论偏推测性，但将格点 Green 函数与 paramodular forms 联系的框架有潜在深度。

5. **整体写作水平较好。** 数学符号规范，结构清晰。

### 缺点

1. **核心恒等式非原创，且未引用先前来源。** 这是最关键的问题。$c_n = \binom{2n}{n} D_n$ 已记录在 OEIS 中，论文未承认这一点。

2. **命题 3.2 的证明有错误。** 缺少 $z^n$ 因子、不必要的重正化、交换求和积分的合理性未证明（虽然实际上可以正确证明）。

3. **ODE 变量 $u = z^2$ 是错误的。** 应为 $u = 64z$。

4. **无图形。** 计算论文应当有图表辅助展示。

5. **可重复性不足。** 无代码、无补充数据文件。

6. **PSLQ 参数讨论不够精确。** 精度范围 "400–800 位" 对某些测试可能不足。

## 五、致作者总结意见

Zhou 先生：

本文选题好、计算扎实，PSLQ 排除程序和高精度数值是有价值的贡献。但当前版本存在若干必须解决的问题：

**必须修改（否则无法接受）：**

1. **诚实地标注 $c_n = \binom{2n}{n} D_n$ 的先前文献。** 此恒等式已记录在 OEIS A002896 中，不应以新定理的形式提出。建议重新定位为：本文的贡献是为此恒等式提供递推证明、揭示 Hadamard 乘积的 CY 算子解读、并推导积分表示。

2. **修正方程 (1.3) 的变量定义。** $u = z^2$ 应改为 $u = 64z$（或文中使用的正确变换）。

3. **重写命题 3.2 的证明。** 补上 $(1/64)^n$ 因子，移除不必要的重正化 $G_D$，直接使用 Domb 生成函数 $g_D(z)$，并用控制收敛定理严格证明交换。

4. **细化 PSLQ 参数报告。** 为每类测试明确给出使用的精度、向量维度、系数界，以及最小关系范数下界。

**强烈建议修改：**

5. 补充至少 2–3 幅图形（被积函数衰减、Richardson 收敛、$G_d(0)$ vs $d$ 趋势）。
6. 拆分 [BBB2008] 参考文献，修正 [AESZ2010] 年份。
7. 补充 OEIS、Koutschan、Broadhurst 等缺失引用。
8. 补充计算代码（作为补充材料或在线仓库）。

如果上述问题得到妥善处理，本文可以成为一篇关于 $d = 4$ 格点 Green 函数的有价值的计算/分析文章。PSLQ 排除表和高精度数值本身就值得发表，但论文需要在原创性声明上做到诚实，在数学细节上做到严谨。
