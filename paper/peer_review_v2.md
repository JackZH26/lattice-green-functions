### 1.3 🔴 方程 (1.1) 和 (2.2): Bessel 积分公式缺少因子 d — 新发现！

**这是本次评审发现的最严重问题，初稿评审未检出。**

论文声明两个等式：
- (1.1): $G_d(0) = \int_{[0,\pi]^d} d^d\theta / (\pi^d(d - \sum\cos\theta_j))$
- (text): $G_d(0) = P_d(1/(2d)^2) = \sum c_n/(2d)^{2n}$

**这两个定义不一致。** 通过 EGF 推导：

I₀(2x)^d = Σ c_n x^{2n}/(2n)! （OEIS A039699 EGF，数值验证 n=0,1,2 通过）

代入 I₀(t) = I₀(2·t/2)，得 I₀(t)^d = Σ c_n t^{2n}/(4^n(2n)!)

因此：
$$\int_0^\infty e^{-dt} I_0(t)^d\,dt = \frac{1}{d}\sum_{n=0}^\infty \frac{c_n}{(2d)^{2n}} = \frac{1}{d}\,P_d\!\left(\frac{1}{(2d)^2}\right)$$

**数值验证（d=4）：**
- Bessel 积分 $\int_0^\infty e^{-4t} I_0(t)^4\,dt \approx 0.30987$
- $P_4(1/64) = \sum c_n/64^n \approx 1.23947$
- 比值 $P_4(1/64) / \text{Bessel} = 4.000$（精确到数值误差）

**概率论推导确认：**
$G_d(0) = \sum P(X_{2n}=0) = \int_{[-\pi,\pi]^d}\frac{d^d\theta}{(2\pi)^d}\cdot\frac{1}{1-\varphi(\theta)}$

其中 $\varphi(\theta) = \frac{1}{d}\sum\cos\theta_j$，故 $1/(1-\varphi) = d/(d-\sum\cos\theta_j)$。

$$G_d(0) = d\int_{[0,\pi]^d}\frac{d^d\theta}{\pi^d(d-\sum\cos\theta_j)} = d\int_0^\infty e^{-dt} I_0(t)^d\,dt$$

**正确公式（二选一）：**
- 选项A：$G_d(0) = d\int_{[0,\pi]^d}\frac{d^d\theta}{\pi^d(d-\sum\cos\theta_j)}$ （在方程 1.1 和 2.2 中添加因子 $d$）
- 选项B：定义 $G_d(0)$ 为积分本身，则 $G_d(0) = P_d(1/(2d)^2)/d$（修改正文中 $G_d(0) = P_d(1/(2d)^2)$ 的声明）

**影响评估：** 此错误不影响任何计算结果（实际计算使用的是 $P_d(1/(2d)^2)$ 值）、不影响 Domb 分解、不影响 PSLQ 排除。仅影响公式表述的数学正确性。

### 致作者

修订版在原创性声明和证明严格性方面有显著改善。命题 3.2 的重写尤其出色——控制收敛定理的使用正确且充分。

但 Bessel 积分公式（方程 1.1 和 2.2）缺少因子 $d$ 是一个不可忽视的数学错误，尽管它不影响任何计算结果。这是一个 well-known convention confusion：期望回访次数 $\sum c_n/(2d)^{2n} = d \times \text{(Fourier 积分)}$。修复方法是在方程 (1.1) 和 (2.2) 中添加因子 $d$，或者修改 $G_d(0) = P_d(1/(2d)^2)$ 为 $G_d(0) = P_d(1/(2d)^2)/d$。
